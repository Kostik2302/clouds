## Описание встретившихся сервисов

### Amazon EC2

&emsp; Amazon EC2 - это сервис, который позволяет вам арендовать виртуальные компьютеры в облаке. Вы можете выбрать тип виртуального компьютера, количество вычислительных ресурсов, памяти и хранилища, которое вам нужно, и платить только за то, что используете.

### Amazon DynamoDB

&emsp; Amazon DynamoDB - это полностью управляемая база данных NoSQL, которая предназначена для хранения больших объемов данных с высокой скоростью и масштабируемостью. Она идеально подходит для приложений, которым требуется обрабатывать большое количество запросов в секунду.

### AWS Database Migration Service

&emsp; AWS Database Migration Service - это сервис, который помогает вам перенести ваши данные из локальной базы данных в облако. Он поддерживает миграцию между более чем 20 различными типами баз данных.

### AWS Device Farm

&emsp; AWS Device Farm - это сервис, который позволяет вам тестировать ваши приложения на различных устройствах. Вы можете использовать реальные устройства или виртуальные эмуляторы.

### AWS IoT Device Management

&emsp; AWS IoT Device Management - это сервис, который помогает вам управлять устройствами Интернета вещей (IoT). Он позволяет вам регистрировать, отслеживать и управлять устройствами IoT в масштабах.

### Amazon Translate

&emsp; Amazon Translate - это сервис, который позволяет вам переводить текст с одного языка на другой. Он поддерживает более 100 языков.

### Amazon Transcribe

&emsp; Amazon Transcribe - это сервис, который позволяет вам преобразовывать аудио в текст. Он поддерживает более 20 языков.

### AWS CloudHSM

&emsp; AWS CloudHSM - это сервис, который позволяет вам безопасно хранить и управлять своими ключами шифрования. Он соответствует требованиям FIPS 140-2 уровня 3.

### AWS CodeBuild

&emsp; AWS CodeBuild - это сервис, который позволяет вам автоматически компилировать, тестировать и создавать пакеты программного обеспечения. Он избавляет вас от необходимости управлять своими собственными серверами сборки.

### Amazon Comprehend

&emsp; Amazon Comprehend - это сервис, который позволяет вам анализировать текстовые данные. Он может извлекать из текста ключевые фразы, языковую окраску, настроения и другие сведения.

### Amazon Comprehend Medical

&emsp; Amazon Comprehend Medical - это сервис, который позволяет вам анализировать медицинские тексты. Он может извлекать из медицинских текстов информацию о пациентах, диагнозах, процедурах и других сведениях.

### AWS Backup

&emsp; AWS Backup - это сервис, который позволяет вам автоматически резервировать данные в облаке. Он поддерживает резервное копирование данных из различных сервисов AWS, таких как Amazon EBS, Amazon RDS и Amazon DynamoDB.
